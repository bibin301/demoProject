1 What is Meeting Minutes?

Meeting Minutes is a web-based tool that helps to capture and manage minutes of a meeting. Actions from the meeting can be recorded and assigned to attendees and people on the distribution list along with target dates.

Features

Meeting Minutes is a web based tool that provides an automated framework to capture and track minutes of meeting.
  1. • Web based solution: No additional software management is required at client (user) end other than the standard web browser.
  2. • Paperless solution: Provides a paperless solution including sending email's on the minutes captured.
  3. • Controlled access: The minutes of meetings are allowed based on attendee. Hence, the tool will take care of allowing only the needed minutes to each user.
  4. • Consolidated Actions: Provides the list of all the actions for which the current user has to act upon including the status. The user can further act on the action/s and update the status.
  5. • Emailing the minutes: Provision for the minutes owner (author) to send mails to all the attendees, thereby providing a copy of the minutes.
  6. • Integrated to Windows authentication framework: User's need not maintain separate username/password for accessing the application. The same username (Windows username/password) is used for authentication.

Usage requirement

The following are the criteria needed for using the meeting minutes tool
  1. • Need a web browser (currently tested only for Internet Explorer 9 and above) with JavaScript enabled.
  2. • Need to be part of the Assystem's network.
  3. • Permission settings to access the Meeting minutes. The administrator to provide the access is with "TIG team" at India. Please contact the TIG team in-case the access (link) is not available for your user-id.

Creating New Meeting Minutes

This tool allows you to create minutes in two ways:
  1. 1. Creating minutes for a new meeting (that is there is no precedent meeting on the same topic).
  2. 2. Creating minutes for follow-on meetings.
    3. To understand the above Point 2, here is an example: Let us say a team of people meet for a meeting called "Finance Monthly Review" for the first time in the month of June. So, the minutes will be created under the department "Finance" and can have a meeting title "Monthly Review". So, the minutes created here does not have precedence as this is being created for the first time. When a meeting is held the following month for the same topic to discuss finance status, another minutes would be created that is a follow-up to the previous one. By doing this, the actions (along with a few other items) from the previous meeting would be carried over. Now, if this team meets again (same or different set of people) for discussing status of projects, a new Meeting Minutes could be created under the "Operations" department with the title "Monthly Review". Note that this minutes and the one created under "Finance" are two different minutes. The following images will show how to create new minutes and how to create a continuum.

Working with the Tool

To create a new Meeting Minutes, click "Create Meeting" link (as shown in Figure-3), which will lead you to the page as in Figure-4.

Meeting title

You can either type in the title or select one from the drop-down list (which can be got by pressing the space-bar once). The title cannot be the same as an existing title for that department.

Date, Time and Venue

Similarly, the meeting date, time & venue can be filled. Note that Department, Title and Venue are mandatory fields.

Attendees and distribution

Attendees and distribution list can be built by selecting staff from the "Employees" list and clicking on the respective addition buttons.

External attendees

If there are people who are part of the meeting or need to get copies of the minutes and are not part of the Organization, then they can be added by typing in their name in the "External Attendee" and their corresponding email id.

Agenda

Add the agenda reference, if any.

Minutes

After this, you could start recording the minutes and any associated actions.

Actions

To add an action, fill in the "Issue" box, select the Target Date from the Calendar provided, select the owner of the Action from the drop-down box and click on the "Assign" button. This will produce a new action in a list below. Unlimited actions can be added in this manner. In the end (or at regular intervals) click on "Save Meeting". This will save the data and exit out of the page.

View

To get back to the same page, click on "View" for that meeting (see Figure-5). This time, at the end of the page, you will notice a "Send Meeting" button. Clicking on this will send all the data for that meeting minutes to all the people in the Attendees, Distribution & External Attendees' list via email.

Continuation meeting minutes

Continuation meeting minutes can be created only when the previous minutes have been sent (using the "Send Meeting" button). Continuation Minutes will contain "Outstanding Actions" list. These actions will be in 'Open' state to start with. The "Action Taken" text box can be filled in either by the Author or the Owner of the Action. While the Author can update this text box while in this page, the section below will describe how the Owner can update. However, only the Author can 'Close' the action by selecting 'Close' from the drop-down box in the Status column. Once the action is 'Closed' the colour of that row darkens. This action is reversible by the Author till a new Meeting Minutes is created.
