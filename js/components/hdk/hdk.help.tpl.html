Create New Ticket

For creating new tickets click on "My Tickets" tab, ticket creation form appears, this can be used for creating the form.

  • • Title: Provide the brief description of the problem.
  • • Category: Provide the appropriate category. This is broad functional classification like Sys Admin, Appraisal, Canteen, Leave, Travel ...
  • • Area: Select Area Based on Category.
  • • Description: Provide the detailed problem so that it could help the ticket-catgeory owner to understand the problem.
  • Click on "Create" button after adding all the required fields. Mail will be sent to the ticket-catgeory owner.

NOTE:

  • 1.Attachments are not supported
  • 2.QMS Forms and processes are still to be followed as required.

View / Update Tickets

Once you have tickets created, you will see them automatically listed in the table below.

  • • Each row is a ticket which has been raised by you
  • • Each row indicates the brief description including current status and a link More link to see more details .
  • • Clicking on More link will show the Ticket tree as well as status of the selected ticket
  • • You can Close / Re-open any ticket which has been RESOLVED by the ticket-category owner.
